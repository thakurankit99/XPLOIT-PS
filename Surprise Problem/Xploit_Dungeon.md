# 🕹️ XPLOIT — The Dungeon Challenge
### XPECTO 2026 · IIT Mandi Tech Fest

---

## What You've Been Given

A compiled game binary — **`xploit_dungeon.exe`**

It doesn't work the way it should. Your job is to figure out why, fix it, and prove it.

---

## Task

The game has multiple stages. Every stage has a reason it cannot be completed. Some reasons are obvious. Some aren't.

**Make the game completable. All stages. Start to finish.**

How you get there is entirely up to you.

---

## What to Submit

```
<teamname>_xploit.zip
├── xploit_patched.exe
└── REPORT.md
```

**`xploit_patched.exe`** — a working build of the game where every stage can be completed.

**`REPORT.md`** — for each stage you fixed:
- What was broken
- How you found it
- The exact change you made
- How you confirmed it works

---

## Scoring

| | Weight |
|---|---|
| Technical depth of your patch | 80% |
| Stages cleared | 10% |
| Report clarity & reproducibility | 10% |

Creativity of the fix counts. A precise, minimal patch scores higher than a brute-force rewrite.
